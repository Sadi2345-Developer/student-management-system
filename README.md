# 🎓 Student Management System

## 📌 Project Overview

The **Student Management System** is a full-stack MERN application developed to manage student records efficiently. Users can add, update, view, and delete student data through a clean and responsive user interface.

The frontend of this project is built using **React + Vite** and integrated with backend APIs using **Axios**. The backend is developed with **Node.js** and **Express.js**, while **MongoDB** is used as the database. The database is connected and deployed live using **MongoDB Atlas**.

## ✨ Features

- ➕ Add new students
- 📝 Update student information
- ❌ Delete student records
- 📋 View all students data
- 📱 Responsive user interface
- 🔄 Full CRUD Operations
- 🌐 REST API Integration

---

## 🧑‍🎓 Student Functionalities

- Users can add student data through the **Add Student** form.
- Student records can be edited using a separate update form.
- Users can delete student records after confirmation.
- All student data is displayed dynamically from MongoDB database.

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Sadi2345-Developer/student-management-system.git
```

---

### 2️⃣ Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

### 3️⃣ Run Frontend

```bash
npm run dev
```

---

### 4️⃣ Install Backend Dependencies

```bash
cd backend
npm install
```

---

### 5️⃣ Run Backend

```bash
npm start
```
